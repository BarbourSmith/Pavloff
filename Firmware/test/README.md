# Test Directory

This directory is for unit tests that can be run with PlatformIO's testing framework.

## Usage

Place your test files here with the naming convention `test_*.cpp`:
```
test/
├── test_main.cpp
└── test_sensors.cpp
```

Run tests with:
```bash
pio test
```

## Documentation

For more information on PlatformIO testing:
